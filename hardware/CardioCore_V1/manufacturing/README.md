# CardioCore V1 — Manufacturing Outputs

> **Research / prototyping / education only.** CardioCore V1 is **not** a certified medical
> device and is not intended for diagnosis, treatment, patient monitoring, emergency use, or
> any clinical decision-making.

This folder holds the fabrication and assembly deliverables generated from the CardioCore V1
PCB design once the layout is complete.

## Status

**Not started.** No PCB layout exists yet, so no manufacturing outputs have been generated.

## Intended Contents

| Artifact | Description |
| --- | --- |
| Gerber files | Copper, soldermask, silkscreen, and board-outline layers (RS-274X). |
| Drill files | Plated/non-plated drill data (Excellon). |
| Pick-and-place / centroid | Component placement (X/Y, rotation, side) for assembly. |
| Assembly drawings | Reference views, designators, fiducials, and notes for PCBA. |
| Fab / assembly notes | Stackup, finish, tolerances, special instructions, and house-specific requirements. |
